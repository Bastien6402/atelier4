Manuel d'utilisation du simulateur

Introduction

Ce manuel vous guide dans l'utilisation du simulateur de vol affiché précédemment.

Instruments de bord

Voici une description des principaux instruments :

  • Horizon artificiel : Permet de visualiser l'attitude de l'avion.
  • Altimètre : Indique l'altitude actuelle.
  • Compas : Affiche la direction suivie.
  • Manette des gaz : Contrôle la puissance des moteurs.

Commandes principales

Les commandes du simulateur peuvent être utilisées via le clavier :

  • Flèche Haut : Monter.
  • Flèche Bas : Descendre.
  • Flèche Gauche : Virer à gauche.
  • Flèche Droite : Virer à droite.

Conclusion
